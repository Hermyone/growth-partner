🌈 介绍

基于 vue3.x + typescript + vite+pinia + element plus + vue-router-next + next.vuex，适配手机、平板、pc 的后台开源免费模板。



## 开发

```bash
# 进入项目目录
cd manager

# 安装依赖
npm install

# 启动服务
npm run dev
```

浏览器访问 http://localhost:9550

## 发布

```bash
# 构建生产环境
npm run build
```
